# Prim's Algorithm (Minimum Spanning Tree) in Python
Prim's algorithm is a greedy algorithm used to find the Minimum Spanning Tree (MST) of a connected, weighted, undirected graph by growing a tree from a starting vertex, adding the cheapest available edge that connects a vertex inside the tree to a vertex outside the tree until all vertices are included. It has a time complexity of O(E log V) where E is the number of edges in the graph and V is the number of vertices in the graph.
